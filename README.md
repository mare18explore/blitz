NFL standings and player stats tracker built with React and ESPN's live API.

## Features
- Live NFL standings for AFC and NFC
- Player stats (coming soon)

## Tech Stack
- React, Vite, React Router
- ESPN public API
- Plain CSS

## Run Locally
npm install
npm run dev
